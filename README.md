## Умное БТИ — тестовые учётные записи

В dev-сборке приложение автоматически заполняет БД базовыми данными (см. `backend/app/db/init_data.py`).

Готовые учётки:

- **Админ**:  
  - email: `admin@example.com`  
  - пароль: `admin123`

- **Исполнитель**:  
  - email: `executor@example.com`  
  - пароль: `executor123`

> Примечание: сидинг срабатывает при старте приложения (создание таблиц и вызов `init_data()` в `backend/app/main.py`). Если данные уже есть, повторный запуск не создаёт дубликаты.
